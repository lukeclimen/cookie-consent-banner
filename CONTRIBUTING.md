# Contributing

Thanks for your interest in contributing! Here's what you need to know.

## Getting Started

1. Fork the repo and create a branch from `main`.
2. Install dependencies: `yarn install`
3. Make your changes, add tests where appropriate.
4. Run the test suite: `yarn test`
5. Open a pull request. If you're fixing a bug, reference the issue number. If you're adding a feature, open an issue first so we can discuss it.

## Reporting Bugs

Open an issue and include what you expected, what actually happened, a minimal reproduction, and your browser/OS.

## Code Style

- TypeScript throughout — avoid `any` where possible.
- No external runtime dependencies.
- Follow [Conventional Commits](https://www.conventionalcommits.org/) for commit messages.
- Run `yarn lint` before pushing.
